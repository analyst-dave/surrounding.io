**Feature: Image Tagging**

### Overview

The `ImageTagger` feature allows users to upload images and automatically generates relevant hashtags based on the image content using AI-powered computer vision analysis. The goal is to simplify the process of tagging images, making it easy for users to earn rewards by uploading high-quality images that are likely to be auto-tagged with many relevant hashtags.

### Requirements

* Users can upload an image file (e.g., JPEG, PNG) from their device
* After uploading, a "Generate Tags" button appears, allowing the user to request AI-powered tag generation
* The algorithm analyzes the uploaded image and generates a list of relevant hashtags based on its content (e.g., objects detected in the image)
* The generated tags are displayed below the button with brief descriptions
* Users can earn rewards by uploading images that are likely to be auto-tagged with many relevant hashtags

### Design

The `ImageTagger` component consists of three main parts:

1. **Uploader**: A file input field where users can upload their image files.
2. **Auto-Tagging Section**: After uploading the image, a "Generate Tags" button appears. When clicked, our AI-powered algorithm analyzes the uploaded image and generates relevant hashtags based on its content.
3. **Reward System**: Displays the total number of auto-generated tags and rewards users for earning more tags.

### Implementation Details

1. Uploading the image file using a file input field
2. Loading the image into TensorFlow.js using the `tf.loadImage()` function
3. Generating hashtags using an AI-powered algorithm (e.g., object detection, scene understanding)
4. Displaying the generated tags below the "Generate Tags" button with brief descriptions
